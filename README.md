 Project Overview :
 
The Microfinance Loan Management System is a Java console-based application developed using JDBC to manage borrower registration, loan creation, loan disbursement, installment calculation, and repayment tracking.
It simulates a real-world microfinance workflow using a relational database.

Features :

 Borrower registration
 Loan application and creation
 Loan disbursement
 Installment (EMI) calculation
 Repayment tracking
 Console-based interaction
 Database connectivity using JDBC

Technologies Used :

Programming Language: Java
Database: MySQL (or any JDBC-supported DB)
Connectivity: JDBC
IDE: Eclipse / IntelliJ IDEA
Java Version: JDK 8 or above
